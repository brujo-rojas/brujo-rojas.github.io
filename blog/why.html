# 🚧 ¿Por que hacer un blog ahora? WIP

Desde hace mucho tiempo he tenido en mente hacer mi propia página web, perfil online, blog, galería, o lo que fuese, sin mayores éxitos, aunque no me quede por falta de intentos, pero ¿Para qué? y ¿Por que no ha funcionado?

Les presento mi ultimo intento, una página personal.

# Para decir quien soy

Y no solo me refiero a un perfil como el de una red social (que las considerare para fines distintos, stalking jaja), sino que tener un sitio donde juntar temas de interés, notas para mi mismo, apuntes y demás, accesible desde cualquier parte, editable fácilmente, y barato (en tiempo y dinero).

# Para crear post de desarrollo

Como desarrollador Frontend, suelo leer muchos post en internet, muchísimos, e intento siempre aprender tecnologías nuevas, herramientas y lenguajes, y eso me hace tener ganas de compartir este mismo conocimiento de alguna forma, escribirlo para mí y para el resto.

# Para tener una galería online

¿Que clase de página es esta que tiene post de desarrollo y galería de arte? Es una página personal, insisto. Quiero tener un link donde pueda compartir mi arte de una manera cómoda, y no solo como un post de instagram o algo así, una manera limpia de compartir imágenes.

# Para centralizarme en internet

Con tantos perfiles en internet, como en linkedin, facebook, twitter, instagram, whatsapp, etc etc etc. ¿Qué puedo compartir cuando quiero mostrar cosas mías? cada una de esas tiene contextos y finalidades muy distintas, y quiero tener una opción más controlada.

# Para enseñarme a hacer cosas

Y lo digo así, Enseñarme a mi mismo a hacer cosas, comenzando por "como hacer un blog personal", siendo el primer post, este, donde me justifico. La mejor manera de aprender es enseñar, ¿y si me enseño a mi mismo? mucho mejor, pero ¿como saber si me enseño bien? Pretendo que estos "respaldos" serán mi propio control, donde mi yo del futuro me juzgara y editara todo esto.

# Para escribir más

No suelo escribir, aunque uso mucho el teclado, lo que más hago es escribir código o chatear. Ahora tengo una buena razón para escribir más, tanto en computador como en papel al hacer borradores.

# Conclusión

Como todo experimento, tengo muchas expectativas, pero nada muy claro, así que solo sera cosa de tiempo para conocer los resultados y aprender de ello.
